    <p>安卓音乐播放框架使用。</p>
    <p>MediaBrowser MediaController MediaSession</p>
    <p>从BaseActivity创建MediaBrowserCompat时传递MusicService，创建本地服务，该服务继承

    数据，MediaBrowserFragment用于显示，PlaybackControlsFragment用于播放控制。</p>
